# Zerith Studio Official Website

A clean, futuristic landing page for the **Zerith Studio** 3D modeling engine. This repository contains a dark‑mode, responsive site built with plain HTML, CSS, and minimal JavaScript. The site is optimized for performance and ready for deployment on GitHub Pages, Vercel, or Netlify.

---

## 🚀 Project Goal
Design a premium, high‑tech landing experience that reflects the brand’s electric purple aesthetic and minimalist dark theme. The page should feel modern yet lightweight—much like the software itself.

## 🎨 Brand Identity
- **Primary color:** Electric Purple (#8A3CFF)
- **Accent glow:** #B155FF
- **Background:** Deep black / dark graphite (#0E0E12)
- **Style:** Futuristic, minimal, premium
- **Theme:** Dark mode only
- **Favicon:** `favicon.ico` (use a `.ico` file for maximum browser compatibility)
  ```html
  <link rel="icon" type="image/x-icon" href="favicon.ico">
  ```

## 🛠 Technology Stack
- HTML5 (semantic markup)
- CSS3 (flexbox/grid, scoped animations)
- Vanilla JavaScript for interactive effects
- Responsive layout for desktop, tablet, and mobile

---

## 📐 Page Structure
1. **Hero Section**
   - Bold logo, headline «Precision Meets Creation.»
   - Subheadline and dual CTA buttons
   - Subtle animated purple glow background

2. **Features Section**
   - Four cards with icons
   - Hybrid Modeling, Lightweight Performance, Custom `.zrs`, Plugin Architecture
   - Hover glow, smooth transitions, glassmorphism style

3. **Preview Section**
   - Large UI mockup
   - Descriptions of real‑time viewport, GPU acceleration, clean interface

4. **Performance Section**
   - Animated counters showcasing speed, RAM usage, GPU rendering

5. **Download Section**
   - Prominent call‑to‑action, system requirements, Windows 10/11 badge

6. **Footer**
   - GitHub link, contact placeholder, copyright, minimal layout

---

## ✅ Design Requirements
- Smooth scrolling and fade‑in animations
- Purple accent lines and neon glow effects
- Google Fonts (Poppins or Inter)
- Light weight (no heavy frameworks)
- Semantic markup with ARIA labels for accessibility

## ✨ Extra Visual Details
- Subtle animated gradient background
- Glass effect panels using `rgba` + `backdrop-filter`
- Neon hover glow on buttons
- Clean, uncluttered interface

## 📁 Output Expectations
- `index.html` with navigation, gradient, counters
- `styles.css` with dark glassmorphism and responsive rules
- `script.js` for scroll effects and back‑to‑top button
